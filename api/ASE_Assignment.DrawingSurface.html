Circle(int, bool)

Draws a circle from the pen's current position, with the specified radius.

public void Circle(int radius, bool filled)

Parameters

radius int

The radius of the circle, in pixels.

filled bool

True to draw a filled circle; false to draw only the outline.

Examples

Example 1:

circle 50, true

Creates a filled circle with a diameter of 100 pixels.

Example 2:

circle 100, false

Creates an outlined circle with a diameter of 200 pixels and an edge width matching the pen's width.

DrawTo(int, int)

Draws a line from current pen position to desired pen position.
Updates pen position when finished.

public void DrawTo(int x, int y)

Parameters

x int

Desired final X position of pen

y int

Desired final Y position of pen

Examples


Example 1
drawto 100,100
Will draw a line to position 100,100 and update Xpos and Ypos
All new drawing will be done with the pen from this position unless moved.

Example 2
drawto 100000,100000
Anything beyond bounds will result in an error message.

Exceptions

CanvasException

Drawing beyond DrawingCanvas boundaries
Check Canvas_Width and Canvas_Height.
